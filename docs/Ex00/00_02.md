# Task 02: Sign in as Lab User One

You'll need to sign in as **user1@@lab.Variable(userDomain)** throughout the lab, so should setup MFA on the account in advance.

{: .note }
If you're performing these steps usinmg your own tenant, you'll need to create yuor own credentials for *Lab User One*.

{: .warning }
> Due to a Microsoft policy, you'll need to perform these steps in a personal browser window, **outside the lab environment**.

1. On your personal machine, outside the lab VM, go to [portal.azure.com](https://www.portal.azure.com/).

1. Sign in with **Lab User One**:

    | Item | Value |
    |:---------|:---------|
    | Username   | `user1@@lab.Variable(userDomain)`  |
    | Password  | `P@ssw0rd!2025`  |

1. Configure MFA on the account.

1. Once configured, close the InPrivate browser window.
